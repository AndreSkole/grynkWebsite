What is Grynk?

Grynk is "The Scripting Language for the Modern Internet" ‚Äì a fast, expressive language that prioritizes readability, productivity, and powerful built-in tools. It is designed for web APIs, AI agents, automation, real-time tasks, and modern desktop GUIs with GxGUI.

  • ‚óÜ Feels fast like Rust but readable like Python.
  • ‚óÜ Zero imports ‚Äì everything through the global gx object.
  • ‚óÜ Built-in CLI, REPL, project scaffolding, and VS Code extension.
  • ‚óÜ Eliminates unnecessary boilerplate and symbol noise ‚Äì only what you need to build.

Built for the modern stack

Grynk ships with a comprehensive toolbox directly on gx: HTTP client, AI helpers, filesystem, crypto, time utilities, data helpers, terminal tools, system commands and more ‚Äì with zero import statements.

With the built-in GxGUI engine, you can build desktop applications in a handful of lines, powered by CustomTkinter for a modern look out of the box.

Install easily via an official Windows installer, or build from source / use the standalone binary on Linux and macOS.

Install & Get Started

The recommended way to install Grynk is via the official Windows installer. It automatically sets up the CLI tool and installs the VS Code extension so you get the Grynk icon and syntax highlighting right away.

  1. Download Grynk_Windows_Installer.exe using the button below.
  2. Run the installer and click through the wizard.
  3. Open any terminal and run grynk repl or grynk run app.grk.
Download for Windows (.exe)

Linux & macOS

You have two ways to install Grynk on Linux (and similarly on macOS):

  • Option 1 ‚Äì Standalone binary (easiest)
    1. Go to the Releases page and download the grynk file (the one without an extension).
    2. Make it executable and move it into your local bin folder so you can run it from anywhere.
  • Option 2 ‚Äì Install from source
    1. Clone the Grynk repository to your machine.
    2. Run the provided setup.sh script to automatically install dependencies and create a global grynk command.

Once installed, you can run:

grynk help, grynk repl, grynk run app.grk ‚Äì exactly like on Windows.

Create & run your first script 
# Create a new file
echo "# Run a quick AI prompt and fetch from the web!" > app.grk

# Run your script
grynk run app.grk

# Start the REPL
grynk repl

# Scaffold a new project
grynk new myproject
Install from standalone binary (Linux) 
chmod +x grynk
sudo mv grynk /usr/local/bin/

# Now you can run:
grynk help
grynk repl
grynk run app.grk

API Quick Reference

A selection of the most important gx.* methods. Type to filter.

Method Category Description
gx.get(url) Web / HTTP Fetch JSON or text from an HTTP endpoint.
gx.post(url, data) Web / HTTP Send POST requests with body data.
gx.scrape(url, selector) Web / Scraping Scrape HTML and extract elements.
gx.ai(prompt) AI Run a free-form AI prompt and return text.
gx.ai_summarize(text) AI Summarize long text using AI.
gx.ai_code(code) AI Get AI assistance for code.
gx.uuid() Crypto Generate a random UUID.
gx.sha256(text) Crypto Compute a SHA-256 hash of the text.
gx.hash(text) Crypto Hash helper (implementation-defined).
gx.read(path) Filesystem Read a file as text.
gx.write(path, data) Filesystem Write data to a file.
gx.read_json(path) Filesystem Read and parse a JSON file.
gx.now() Time Get current time.
gx.sleep(ms) Time Pause execution for given milliseconds.
gx.sort(list) Data Sort a list.
gx.unique(list) Data Remove duplicate values from a list.
gx.map(fn) Data Map a function over a list.
gx.filter(fn) Data Filter a list by predicate.
gx.banner(text, color) Terminal Print a stylized banner with a color.
gx.table(data) Terminal Render data as a terminal table.
gx.clear() Terminal Clear the terminal screen.
gx.run(cmd) System Run a system command.
gx.env(name) System Read an environment variable.
gx.platform() System Get information about the current platform.
gx.window(title, w, h) GxGUI Create a new GUI window.
gx.label(text) GxGUI Add a label to the window.
gx.button(text, fn) GxGUI Add a button that runs a function on click.
gx.entry(placeholder) GxGUI Add a text input field.
gx.gui_run() GxGUI Start the GUI event loop.
